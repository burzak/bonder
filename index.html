Bonder

The object oriented PHP framework

View on GitHub Download .zip Download .tar.gz

Bonder is a PHP Framework mainly focused on web application backend development. Its main focus is relying on Object Oriented Programming (OOP) structures and techniques, allowing for easy modularization and extension. This project was started due to, even though there are great php frameworks out there, none seemed to completely embrace decoupling and OOP.

Getting started

Installation

We embrace the use of Composer for dependency management in PHP, and as such, this is our preferred method of distribution. Simply add

"require": { ... "getbonder/bonder": "dev-master", ... }

to your composer.json file.

Configure your web server

Bonder is meant to be used with routed URLs. As such, you need to configure your web server for it.

Nginx + PHP-FPM

Add the following server to your nginx configuration.

`server { listen 80; server_name myapplication.domain.com;

root /var/www/myapplication/public_scripts;

location / { fastcgi_index launcher.php; fastcgi_pass 127.0.0.1:9000; include fastcgi_params; fastcgi_param SCRIPT_FILENAME $document_root/launcher.php; fastcgi_param SCRIPT_NAME /launcher.php; } }`

Remember to configure your php-fpm to listen on port 9000, or use fastcgi_pass unix:/var/run/php5-fpm.sock; if you are using sockets.

Launcher script

Now create a public_scripts folder in your application folder (assumed here in /var/www/myapplication), with a launcher.php file with the following contents:

`<?php include_once "../vendor/autoload.php";

session_start();

$launcher = \Bonder\LauncherFactory::getStandardHttp( include "../config/resources.php", include '../config/controllers.php', include '../config/filters.php')->launch(); `

This assumes there is a folder /var/www/myapplication/config with the three files mentioned, which return arrays.
